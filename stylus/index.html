  • WET (Write Everything Twice) Oriented
  • Useless characters
  • Browser dependent
  • Hard to mantain in larger projects

  font: 12px Helvetica, Arial, sans-serif;
}

  -webkit-border-radius: 5px;
  -moz-border-radius: 5px;
  border-radius: 5px;
}

  color: pink;
  /* A bunch of stuff */

/* Same property, different values */
.button {
  -webkit-border-radius: 5px;
  -moz-border-radius: 5px;
  border-radius: 5px;
}

  -webkit-border-radius: 15px;
  -moz-border-radius: 15px;
  border-radius: 15px;
}

  color: white;

Simple grid

v = zero one two three four five six \
  seven eight nine ten eleven twelve
columns = 12
plural(i)
  i > 1 ? 'columns' : 'column'
for i in 1..12
  .{v[i]}-{plural(i)}
     width: (100 * i / columns)%
     display: inline-block
